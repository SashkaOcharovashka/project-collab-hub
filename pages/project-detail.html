Разработка CRM системы для управления клиентами

Категория: Разработка
Опубликован: 15 марта 2026
Просмотров: 234

Описание проекта

Требуется разработать CRM систему для управления клиентами и задачами. Основной функционал:

  • Ведение базы клиентов с историей взаимодействий
  • Учет звонков и интеграция с телефонией (через API)
  • Постановка и отслеживание задач
  • Отчеты и аналитика
  • Интеграция с email-рассылками
  • Ролевая модель доступа (админ, менеджер, сотрудник)

Проект должен быть выполнен на PHP (Laravel) с использованием MySQL. Интерфейс - современный, адаптивный. Требуется разработать как веб-версию, так и API для мобильного приложения.

Требуемые навыки

PHP Laravel MySQL JavaScript REST API Git
